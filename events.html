Events Syntax

load event - raises(fires) a notification when page is completely loaded in browser - unload event Must intercept events; event handler (listener) -" connecting code to event" -to preform task/event -JavaScript primarily event driven-most common user event is mouse events event handler inside tags - events: mouse, keyboard, progression, form, mutation, touch, error Connecting Code to Events - [use of objects/events} - 3-Ways: Assigning HTML Attributes, Assigning Objects special properties, Calling Objects special Methods Handling events via HTML attributes: [click Event fires when clicked] - Click Me clickme/onclick target property - target of the event handling events via object properties - document.getElementById() Method - linkClick() Standard Event Model - addEventListener() - removeEventListener() - API - application programming interface HTML Forms - client side processing information within the browser of grouping HTML interaction elements,

- start and end form; number of attributes(action)
- create HTML Form Element Object - Can Access in two Ways: Name- document.myForm, document Objects form collective property

Events Properties

onchange - An HTML element has been changed onclick - The user clicks an HTML element onmouseover - The user moves the mouse over an HTML element onmouseout - The user moves the mouse away from an HTML element onkeydown - The user pushes a keyboard key onload - The browser has finished loading the page Keep the number of elements in the HTML DOM small. This will always improve page loading, and speed up rendering (page display), especially on smaller devices.
