1.2 Operators

Operators are used in JavaScript to preform many different kinds of operations with variables; all operators are of different types.
These different Operators all work in this similar way in JavaScript; with both the Operators and Operands:
Operator: Symbol used to act upon an item, an expression or a variable.
Operand: Is a variable that operators act upon.
The operation (to be performed between the two operands) is defined by an operator; this is the basis of Operators. The many different types of Operators can be shown in the subsections below and work:.

Arithmetic Operators

References and Information: https://www.w3schools.com/js/js_arithmetic.asp

Operators perform arithmetic on numbers(or literals/variables).

  • + - Addition (Concatenation Operator)
  • - - Subtraction (Unary Operator)
  • / - Division
  • * - Multiplication
  • % - Modulus
  • ++ - Increment(Counts up 1)
  • -- - Decrement(Counts down 1)

Arithmetic Operators Syntax

These Operators perform arithmetic on numbers(or literals/variables).

4 + 5; //result is 9. Or Concatenate Strings - var orStrings = "One String," + " Another String added.";
var aVariable = 12 - numberVariable; // Or can subtract a string(Unary Operator) var orStrings = "One String," - " Another String added."
var aVariable = numberVariable / 5; // Divides
var x * 2; // Multiplies. 20 % 5; // Returns percent values.
var aVariable++ // Variable now Incremented; 20 -- x // Decrement(Counts down 1);

Assignment Operators

References and Information: https://www.w3schools.com/js/js_assignment.asp

These Operators assign values to JavaScript variables.

  • = - Assigns the value of the right side to the left side of the operator(Assignment Operator).
  • += - Adds and assigns to a Variable or DataType.
  • -= - Subtracts and assigns the result to a Variable or DataType.
  • *= - Multiplies and assigns to a Variable or DataType.
  • /= - Divides and assigns to a Variable or DataType.
  • %= - Modulus result and assign to a Variable or DataType.

Assignment Operators Syntax

var canAssign = 50; // The Assignment operator(equals sign) gives canAssign the value of 50.

var addAndAssign = 10 += "100" ;
//results to 10100; ( += ) - Assignment operator can also be used to add(Concatenate) any code/data.

var subAndAssign -= 120;
// Given subAndAssign is numeric, or results may be NaN(Not a Number).

x *= AnyDataVariable; // Multiplied and assigned to aVariableDataValue /= 20; // Divided and assigned to 209 %= percentVariable // Modulus result(percentage) and assign

Comparison and Logical Operators

References and Information: https://www.w3schools.com/js/js_comparisons.asp

Comparison operators return a value of "true" or "false":

  • == - Equal to
  • === - Equal value and equal type
  • != - Not equal
  • !== - Not equal value or not equal type
  • > - Greater than
  • < - Less than
  • >= - Greater than or equal to
  • <= - Less than or equal to
Ternary Operator

Also a conditional operator that assigns a value to a variable based on a condition.

Logical Operators

Are used to determine the logic between variables or values, and can be used together for more complex results.

  • && - And
  • || - Or
  • ! - Not

Comparison and Logical Operators Syntax

4 == 4; // result "true"
10 === "10"; // result "false"
a != x; a !== x; // These operators compare variables, and can be used in many different ways. This example compares; is a != x?
x > y //is x >(greater than) y? 10 < 12; // result "true" x >= y; 10 <= 10;
var compareVar = (age < 18) ? "Too young":"Old enough";

(x > 10) ? "if true":"if false"; // Ternary Operator Compares based on a condition, and can produce a certain result.

(1 < 10 && 10 > 1) // result "true"; in this case And is used to evaluate an expression. If once condition inside this statement were false, the result is "false".
(10 > 5 || y > 5) // result "true"; here using Or to determine Logic in a statement.
!(x == y) // result "true"; and here even though x does not equal y the Not operators logic makes this result not "false" but "true"(Reverse Logic).

Operator Properties

Operators have their own function as shown, but aside from this operators also have what is called Operator Precedence Highest [], (), ! ~ - + ++ --, * / %, + -, == !=, && || ?, = += -= *= /= %=, , Lowest

var PrecedenceVar = (20 - 10) / 2; // the expression in Parentheses, enable the correct equation to be calculated.
