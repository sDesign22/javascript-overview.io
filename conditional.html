1.3 Conditional Statements (if, ...else if, switch)

These statements can be used in Control Flow, using Boolean Data code can make decisions; and depending if a Condition is met in your statement will follow certain actions.

if

Evaluates an expression and automatically converts the result of the expression into a Boolean (true or false). If condition true; statement is executed.

if Statement Syntax

if(statement > statement2) { Executes Code in here; } // This statement compares the conditional(Expression); and executes code inside { } if condition is true.
if(3 > 2) { Executes Code in here; } // This code will execute because 3 is greater that 2; therefore condition is true and executes code.

The properties and methods of if can vary; its properties are defined by the statement itself. The methods using if as mentioned are great, you can change and use statements as you need. Conditional statements are again used with comparison/logical operators for control flow(programming intelligence).

...else if

Else if - is its own Keyword and used with an if statement for an alternate solution to the conditional statement.

...else if Syntax

if (aVariable >= 18) { Code to be executed; } else { the alternate else Code displays; } // This example shows the if condition (aVariable >= 18); if true will execute that statements Code Block, if false then commands inside the else braces will execute.

if (condition1) { Code to be executed if condition1 true; } else if (condition2) { Code to be executed if condition1 is false, and condition2 is true; } else { Code to be executed if first statement conditions are false; } // This example shows if else condition as well, as if condition.

The properties and methods of ...else if can vary also; its properties are defined by the statement itself like if. The methods you can change and use as you need.

Switch

Switch to the code where the case statement matches; Each case in the switch statement means: "If the expression is equal to the value, execute the statement".

Switch Syntax;

switch(expression) {case 1: code block to execute; break; case x: code block to execute; break; default: The default code block to execute; } // Use the switch statement to select one of many code blocks to be executed.

Conditional Methods

Comparison and Logical Operators are often used in Conditional statements to determine equality or difference between variables or values. And with this can produce Control Flow(add logic) to code:

if (1 < 10 && 10 > 1) { if statement using a Logical operator; } // is true so code will execute.

break - Keyword causes the code to jump out of the switch case statement. Without the break statement, code will continue to evaluate the expression even if one case already met the condition.
default - Keyword indicates what the program should do if the expression does not evaluate to one of the cases

switch(expression) {case 1: code block to execute; break; case x: code block to execute; break; default: The default code block to execute; } // This shows our switch example, it has the default keyword so if no cases match can run accordingly.

break and default can be used in other statements as well.
Nested Conditions: Nested one into the other as an outer if statement - witch surrounds an inner if statement if.
