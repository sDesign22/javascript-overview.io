1.9 String Object

Strings Consists of zero or more characters - You can use quotes inside a string, as long as they don't match the quotes surrounding the string.

String Object Syntax

Strings {Objects}; the keyword new - var firstName = new String("John");

String Object Properties

constructor - Returns the string's constructor function.
length - Returns the length of a string.
prototype - Allows you to add properties and methods to an object.

String Object Methods

charAt() - Returns the character at the specified index (position) - user-input
charCodeAt() - Returns the Unicode of the character at the specified index myString.charCodeAt(0) - user-input
concat() - Joins two or more strings, and returns a new joined strings
endsWith() - Checks whether a string ends with specified string/characters
fromCharCode() - Converts Unicode values to characters

myString = String.fromCharCode(65,66,67); - Converts

includes() - Checks whether a string contains the specified string/characters
indexOf() - Returns the position of the first found occurrence of a specified value in a string

myString2 = mySting1.indexOf("Hey"); - (regular expressions) Finding

lastIndexOf() - Returns the position of the last found occurrence of a specified value in a string

var pos = str.lastIndexOf("locate", 15); - Find String in Another String

localeCompare() - Compares two strings in the current locale
match() - Searches a string for a match against a regular expression, and returns the matches
repeat() - Returns a new string with a specified number of copies of an existing string
replace() - Searches a string for a specified value, or regular expression; returns a new string

var n = str.replace("Microsoft", "W3Schools"); - By default, replaces only first match

search() - Searches string for specified value, or regular expression; returns position of match
slice() - Extracts a part of a string and returns a new string

var res = str.slice(7, 13); - Extracting

split() - Splits a string into an array of substrings - txt.split(","); // Split on commas
startsWith() - Checks whether a string begins with specified characters
substr() - Extracts the characters from a string, beginning at a specified start position, and through the specified number of character - Extracting/Copying
substring() - Extracts the characters from a string, between two specified indices

myString.substring(0, 4); - Extracting/Copying

toLocaleLowerCase() - Converts a string to lowercase letters, according to the host's locale
toLocaleUpperCase() - Converts a string to uppercase letters, according to the host's locale
toLowerCase() - Converts a string to lowercase letters - var text2 = text1.toLowerCase();
toUpperCase() - Converts a string to uppercase letters - var text2 = text1.toUpperCase();
toString() - Returns the value of a String object - myVar.toString(); - can use to output numbers from base 2 to base 36.
trim() - Removes whitespace from both ends of a string - can be used to correct user input/text
valueOf() - Returns the primitive value of a String object
