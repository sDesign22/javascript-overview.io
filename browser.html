2.1 Browser Objects

Collection of Objects browser makes available; allows JavaScript to "talk to" the browser.

Window Object

Window Object: Includes Properties - Child objects of the window object are actually PROPERTIES of the window object

Screen Object

Screen Object: Contains a lot of information on display capabilities of client machine - properties: height, width: indicated vertical and horizontal[range in pixels] colorDepth: Tells you number of bit for color on clients screen

History Object

History Object: Keep track of each page user visits; list of pages called History Stack. Also enables to click back and forward with pages, like an Array has a length property- can find out how many pages in stack. has back() and forward() Methods; go() Method - one parameter specifies how far or back you want to go in history stack - history.go(-1);back() are equivalent

Location Object

Location Object: Information on the pages location(URL)- Uniform Resource Locator: (href) - server hosting page - hostname, port number or server connection - port, protocol used - protocol
Feature Detection: Check if feature exists in browser; if (navigator.geolocation) { use geolocation } - truthy of falsely - using conditional statement( 0 ,"" ,null ,undefined ,[] ,false) - all false); else mostly truthy [ if supports] - typeof Operator: to determine "type".

Navigator Object

Navigator Object: Find out witch browser, version and system user has; Browser Sniffing[ feature detection]: find if a browser supports a particular feature
geolocation Object(HTML 5 specification); add to navigator Object - used to find position of device - [permissions security]
getCurrentPosition(): Must pass a call back function
navigator.geolocation.getCurrentPosition(callbackFunction); - Has second parameter for when error occurs: position Parameter: Object that contains earthly positions and altitude [coords property]
- navigator Object- 2 properties useful in identifying browser: appName - returns model of browser, userAgent - bits of info for browser/model(varies from browser to browser)

Document Object

document Object: Gain access to html elements and their properties and methods inside your page. Properties[Array like] - called Collections: forms, images, links[all array - of all elements represented by object in page]
Using document object: Properties and Methods: write(), bgColor - screen color depth[number of colors a users screen supports ex2]
images Collection: // First image stored images[0]: var myImage2 = document.images[1]; links Collection: For each hyper-link element
