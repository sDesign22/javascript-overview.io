1.6. JS Objects General Overview

Javascript is an Object based language and most of what you do involves manipulating Objects; this range expands immensely. Objects are Variables too, but can contain many values.
Objects are a collection of: Properties, Methods, Constructor(s);
Properties: Properties are the Values associated with an object. Methods: Methods are the Actions that can be Performed on the object. Constructor(s): Constructors are used to Create an object

JS Objects Syntax

Define/Create: Using the keyword new Followed by object Constructor(). In this function, we declare the initial properties/methods of the new object, and usually assign them a pre-defined value.

var aObject = new Object("value", 2, anotherValueProperty, "for ref Store.");

// This creates an object and stores values that can be stored for further code reference/use.
Object Literal: Can also create Objects; name:value pairs (name and value separated by a colon) called Properties or Methods.

var anObject = {a:1, asName:"value2", aProperty:3, anotherValue:"Property"}; // This creates an object and stores values that can be stored for further code reference/use. All Objects are properties and methods from prototype

Object Properties

Can access object properties in two ways: 1. objectName.propertyName 2. objectName["propertyName"] - this can also add a property
Object Constructors: For creating many objects of the same "type"(values/parameters); the way to create an "object type", is to use an object constructor function. Object Types (Classes);

Objects of the same type are created by calling the constructor function with the new keyword: var objectVariable = new DataStore("information", 100, "Data", x);

Object Methods

Can assign a Method to a function:

anObject.newProperty = aMethodFunction() { code to be executed; } // This now a method as aMethodFunction().

Accessing Object Methods: Stored in properties as function definitions.

anObject.anObjectMethod // Accessing method without parentheses returns function definition.
this - Keyword refers to a current Object, It is preferred to use this in a method to refer to a current object.

function Object{ this.methodRef(); } // Refers to the current object.
