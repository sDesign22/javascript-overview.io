1.4. Loops (for loop and while loop)

Loops are commands that are executed repeatedly until a condition is met (condition true);
Any loop has three statements, each separated with a semicolon:
1. INITIAL; 2. CONDITION; 3. INCREMENT
Initialization: Used to keep track of loops variable; declared and given value witch is number of execution for loops.
Test condition: Code in statement will keep executing if the test condition is true.
Increment: Variables in test conditions have their values incremented with ++ operator; repeated every loop of code(can also decrease value --).
Block of Code: Code to be executed is inside { Curly Braces; } // Statements can be written repeatedly just using loop.

For Loop Syntax

for (statement 1; statement 2; statement 3) { Can loop on Conditions, or using the i variable can place data into the loop; } // Here we have the syntax basics.

For Loop Methods

For/In Loop: Statement loops through the properties of an object.

for (a in x) { writetxt += x[a]; } // This example shows for loop placing x into a.

For/Of Loop: Loop over data structures that are iterable such as Arrays, Strings, Maps, NodeLists and more.

for (aVariable of varIteration) { code block to be executed; } // Using for varIteration can also be iterated into aVariable.

While Loop

Loops through a block of code as long as a Specified Condition is true. test a condition and keep looping while true.

While Loop Syntax

while (condition) { code block to be executed; i++; } // while a condition is true will execute the code block. Notice that this Statement only has a condition, and the Increment statement is inside of the code.

While Loop Methods

Do/While Loop: Variant of while loop and will execute the code block once, before checking if the condition is true; use if necessary.

do { code block to be executed; i++; } while (condition); // Like while loop, but will execute the code based on while condition, not from while condition.
