1.8. Array Object

Can hold any type of data in each slot - Arrays are written with square brackets. Array items are separated by commas. Data values (Elements) - Organized under a single name.

Array Object Syntax

Using Array literal: Is the easiest way to create a JavaScript Array.

var anArray = [item0, item2, "Can be AnyValue"];

Using new Keyword: The following example also creates an Array, and assigns values to it:

var anArray = new Array(item0, item2, "Can be AnyValue");

Access Elements of an Array:
Access an array element by referring to the index number, indexes start with [0] (Machine Code);
Full array can be accessed by referring to the array name; myArray.
Multidimensional Arrays: Can be represented by arrays of arrays.

var myArray[0] [0]; var myArray[2] [4] // This examples shows a Multidimensional Array (array in array).

Array Object Properties

constructor - Returns the function that created the Array object's prototype.
length - Sets or returns the number of elements in an array.
prototype - Allows you to add properties and methods to an Array object.

Array Object Methods

concat() - Joins two or more arrays, and returns a copy of the joined arrays - Manipulation.
copyWithin() - Copies array elements within the array, to and from specified positions.
entries() - Returns a key/value pair Array Iteration Object.
every() - Checks if every element in an array pass a test - OP.
fill() - Fill the elements in an array with a static value.
filter() - Creates a new array with every element in an array that pass a test - OP.
find() - Returns the value of the first element in an array that pass a test - OP.
findIndex() - Returns the index of the first element in an array that pass a test - OP.
forEach() - Calls a function for each array element - OP.
from() - Creates an array from an object.
includes() - Check if an array contains the specified element.
indexOf() - Search the array for an element and returns its position - Finding/OP.
isArray() - Checks whether an object is an array.
join() - Joins all elements of an array into a string - Converting Array into single string.
keys() - Returns a Array Iteration Object, containing the keys of the original array.
lastIndexOf() - Search the array for an element, starting at the end; returns its position -Find/OP.
map() - Creates new array with the result of calling a function for each array element - OP.
pop() - Removes the last element of an array, and returns that element - Stack.
push() - Adds new elements to the end of an array, and returns the new length - Stack.
reduce() - Reduce the values of an array to a single value (going left-to-right) - OP.
reduceRight() - Reduce the values of an array to a single value (going right-to-left) - OP.
reverse() - Reverses the order of the elements in an array - Reordering.
shift() - Removes the first element of an array, and returns that element - Queue.
slice() - Selects a part of an array, and returns the new array - Copy Array-part/Manipulation.
some() - Checks if any of the elements in an array pass a test - OP.
sort() - Sorts the elements of an array - Reordering.
splice() - Adds/Removes elements from an array - Manipulation.
toString() - Converts an array to a string, and returns the result - Conversion.
unshift() - Adds new elements to the beginning of an array, and returns the new length - Queue.
valueOf() - Returns the primitive value of an array - Conversion.
