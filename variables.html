1.1 Variables

JavaScript variables are containers for storing data values, variables are used to hold and store any Value or Type of Data. Variables are of course a base in JavaScript programming because of this; in the following sections below you can see all different types of variable uses and examples.

General Syntax - var Keyword

The var Keyword creates a Variable; once a variable has been declared it needs a name Identifier for memory store reference, preferably one according to its use in the script (for ease). The First Character Must Be: Letter, Underscore(_) or Dollar Sign($):

var nameExample; // Now an Identified variable called nameExample is declared in the script; it's value is undefined (no assigned value).

Case Sensitivity: In the example above you can also see an example of case sensitivity named as - nameExample. Names in JavaScript Variables are Case Sensitive:

var nameExample; var nameexample; // These are two different variables, even though the name is the same you can see the difference in case sensitivity. nameExample vs all lower case nameexample, these are two different Variables.

Reserved Words: Are names that Cannot be used for declaring/naming Variables. Keywords like var are very common in Javascript to preform different actions, these and other programming Keywords are know as Reserved Words.

var char; // One example of a Reserved Word.
var goto; var this; var typeof; // these are more keywords, these keywords preform all different kinds of functions.

Visit https://www.w3schools.com/js/js_reserved.asp for more on Reserved words.

Variable Methods

You can declare many variables in one statement, with each variable separated by a comma:

var nameExample, nameExample1, a, AnotherVariable; // This creates several different Variables, the var keyword is used only once for creating multiple Variables in this statement.

Assigning Values: Using the equals sign ( = ) you can assign a Value to your Variable, like so: var NamedVariable = ;

Fixed Values: Called Literals are values with an Actual (literal) data-value rather then from a calculation or another variable value.
Numbers: Are written with, or without decimals.

var aNumberValue = 10; var aDecimalNumberValue = 9.99;
// JavaScript can use both Integer and Floating Point number values.

Strings: Written inside of double or single quotes; to assign a string (text) value to a variable you must put quotations around the value.

var aStringValue = "Inside of Double Quotations"; var alsoTextValue = 'Or inside of Single Quotations, like so';
// These two examples are how to write string literal values, Strings are recognized in JavaScript by quotations.

var stringNumberValue = '10'; // Numbers are also treated as strings when inside quotations and not as a number values.

Escape Character: / - Is used to ignore code, as you see with our example comments written after //. Special Characters: The escape character also turns special characters into string characters, in addition to ordinary characters strings can also contain special characters:

var myVar = "This is a test call \"My Test \" "; // "My Test" is returned in the code now, using the \" character. var myVar = "This is a test call \n My Test"; // My Test. This moves a new line in the code or tabulator.

More Special Characters: \b - Backspace, \f - Form Feed, \n - New Line, \r - Carriage Return, \t - Horizontal Tabulator, \v - Vertical Tabulator.

Variable Values: Is data-value from a calculation or another variable value (can change or vary the value).

var aVariableCalculation = x + 10; // Using Concatenation with the plus sign ( + ) this method adds 10 to x in this example.

Concatenation is also used to add strings of code, or even Variables together.

var StringConcatenation = "Hello," + " This is another String."
var openVariable = AssignedVariable; var openVariable = AssignedVariable + AssignedVariable2; // These examples show one variable being assigned to another variable. And two variables(AssignedVariable + AssignedVariable1) being assigned to a variable.

Again any Type or Value of Data can be assigned to a variable to be stored; recalled at anytime in your script for use or reassigned.

var oneVariable = anotherVariable // Here one variable is reassigning as another.

Variable Properties

A Variables properties are its values, how the variable is created is the definition of its properties.
let and const - Variables defined with let or const are not hoisted to the top. Declaring a variable with const cannot be reassigned in the same block.
